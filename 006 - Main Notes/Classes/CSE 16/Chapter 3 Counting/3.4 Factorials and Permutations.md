# Counting

## Factorial
- If ${n}$ is a non-negative integer, then ${n!}$ is the number of lists of length n that can be made from n symbols, without repetition. Thus ${0!}$ = 1 and ${1!}$ = 1. If ${n > 1}$, then ${n!}$ = n(n − 1)(n − 2) · · · 3 · 2 · 1.
